# **Face Mask Detector** 

---

**Face Mask Detector**

In this project my goal is to build a model  to detect whether the person on the webcam is wearing a mask or not. We will train the face mask detector model using Keras and OpenCV.


---
### Dependencies
The only special libraries used in this project are: keras, tensorflow, numpy, and opencv.

You can install the dependencies like this:
```
    pip install keras
    pip install tensorflow
    pip install numpy
    pip install opencv
```

### Dataset
For this  project, there is a copy of the dataset used for training already included. 

### To run the project

```
    python train.py
    python test.py
```
